# GPA Calculator

A simple Python application that calculates a student's average score and assigns a final grade.

## Features

- User input
- Average calculation
- Grade evaluation
- Formatted output

## Concepts Used

- Variables
- Data Types
- Input / Output
- For Loops
- Arithmetic Operators
- If / Elif / Else
- f-Strings

## Example

```text
Please Enter the number of subjects: 3

Enter subject mark 1: 90
Enter subject mark 2: 85
Enter subject mark 3: 95

The GPA is: 90.00 - Grade: Excellent
```
